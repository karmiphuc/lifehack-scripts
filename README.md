# lifehack-scripts

Small scripts that earn their keep on my daily-driver machines. Right now the public collection is mostly Windows batch — BitLocker unlock helpers, shortcuts, and the kind of one-liners that save a few seconds per task but compound across a year.

The bar for adding a script here is simple: it has to be something I've used at least ten times in real life. If it's a one-off or a hack, it stays in a scratch repo.

GPL-3.0.
